# GB4778

## Graph-Break Type
*Short name describing what triggered the graph break*

nested function with closure in output

## Context
*Values or code snippet captured at the break point*

as_python_constant for out_spec

## Explanation
*Explanation of why the graph break was triggered*

Cannot return a nested function with closure from a compiled function. Dynamo cannot reconstruct the function.

## Hints
*Hints on how to resolve the graph break*

- Define the function at module scope instead of inside another function
