# GB4271

## Graph-Break Type
*Short name describing what triggered the graph break*

__class__ assignment on user-defined object

## Context
*Values or code snippet captured at the break point*

object={self}, value={value}

## Explanation
*Explanation of why the graph break was triggered*

Dynamo does not support reassigning __class__ on user-defined objects.

## Hints
*Hints on how to resolve the graph break*

- Move the __class__ assignment outside of the torch.compile region.
