# GB2078

## Graph-Break Type
*Short name describing what triggered the graph break*

Attempted to access unregistered member on an OpaqueObject

## Context
*Values or code snippet captured at the break point*

value={real_obj}, attr={name}

## Explanation
*Explanation of why the graph break was triggered*

Member '{name}' is not registered for this opaque object type.

## Hints
*Hints on how to resolve the graph break*

- Register '{name}' with a MemberType in register_opaque_type(members=...).
