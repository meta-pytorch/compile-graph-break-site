# GB9567

## Graph-Break Type
*Short name describing what triggered the graph break*

Unsupported descriptor on opaque class

## Context
*Values or code snippet captured at the break point*

class={self.value}, attr={name}, descriptor={type_name}

## Explanation
*Explanation of why the graph break was triggered*

The attribute '{name}' is a descriptor of type '{type_name}' which is not supported.

## Hints
*Hints on how to resolve the graph break*

- Only staticmethod, property, and pybind11_static_property are supported.
- Consider accessing this attribute outside of the compiled region.
