# GB0376

## Graph-Break Type
*Short name describing what triggered the graph break*

Call to `torch._dynamo.disable()`

## Context
*Values or code snippet captured at the break point*

Called `torch._dynamo.disable()` with args `{args}`, kwargs `{kwargs}`

## Explanation
*Explanation of why the graph break was triggered*

`torch._dynamo.disable()` was called inside a compiled region. This API disables compilation when used as a decorator or wrapper outside the compiled region.

## Hints
*Hints on how to resolve the graph break*

- Move the `torch._dynamo.disable()` call outside the compiled function and apply it to the function that should run eagerly.
- Use `torch._dynamo.graph_break()` to intentionally insert a graph break at this point.
